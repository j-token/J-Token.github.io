§ 5Web — Abyssal Moonlight

인디 게임 深海月光 (Abyssal Moonlight)의 비공식 팬사이트. Next.js 16 App Router 기반 SSR, 한·영·일 다국어 동적 라우팅, 슬롯 단위로 카피·이미지를 실시간 편집하는 어드민 CMS, PixiJS와 Cubism Core로 구동되는 Live2D 캐릭터 스테이지를 한 코드베이스에 묶었습니다.

Next.js 16 React 19 TypeScript Tailwind v4 PostgreSQL Drizzle ORM better-auth PixiJS · Live2D Full-stack Railway · Docker

주요 시스템

  • i18n Routing — App Router 동적 세그먼트 app/[locale]/…로 한·영·일 SSR 분기, 언어 스위처 + 쿠키 기반 선호도 저장
  • Slot-based CMS — content 테이블 복합 PK (key + locale)로 다국어 카피·마크다운·이미지·JSON을 슬롯 단위로 관리. DB가 비어 있어도 코드의 defaultValues로 자동 폴백
  • Admin Auth — better-auth 이메일·비밀번호 + 세션 쿠키, ADMIN_EMAILS 화이트리스트로 어드민 라우트 가드. 변경 이력은 updatedBy FK로 추적
  • Live2D Stage — PixiJS v7 + pixi-live2d-display-mulmotion. Cubism Core는 라이선스상 npm 재배포 불가이므로 CDN 동적 로드
  • Media Pipeline — 어드민 업로드 → AWS SDK → Railway Storage(S3 호환) → 공개 URL → SSR 다음 요청부터 즉시 반영
  • Deploy — Node 22-Alpine 멀티스테이지 Dockerfile + Next.js standalone 출력, Railway에서 Postgres·Storage 환경변수 자동 주입
